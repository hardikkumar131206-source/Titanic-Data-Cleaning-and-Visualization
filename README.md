# Titanic Data Cleaning & Visualization

A complete data cleaning, feature engineering, exploratory data analysis (EDA), and visualization pipeline built using Python, Pandas, Matplotlib, and Seaborn.

---

## Project Overview

This project analyzes the Titanic dataset and explores passenger survival patterns using:

* Data cleaning
* Missing value handling
* Feature engineering
* Outlier detection
* Statistical analysis
* Professional visualizations

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## Features

### Data Cleaning

* Missing value analysis
* Age imputation using grouped medians
* Embarked value filling
* Duplicate removal

### Feature Engineering

* FamilySize
* IsAlone
* AgeGroup
* FareBand
* HasCabin

### Visualizations

* Survival dashboard
* Correlation heatmaps
* Fare analysis
* Family analysis
* Data quality reports
